# Salary_Census_Data

<b>This project is carried out on the dataset freely avaliable on Kaggle</b>

<b> Features in the dataset include :</b>

* Age
* work_class
* Education
* Education.num
* Marital_status
* Occupation
* Relationship
* Race
* Sex
* Capital_gain
* Capital_loss
* House_per_Week
* Native_Country
* Income

<b> The Goal :</b>

Predict whether income exceeds $50K/yr based on census data
